# 参数说明
前言，首先是参数说明：
adm=管理员操作

```
adm=ban=60　禁言60秒（以60秒为例），如果要永久禁言，干脆设置小于30秒或大于1年的时间，设置0也行。
adm=warn=1　加一次警告，三次则踢出并封禁。
adm=kick=60　踢出并封禁60秒，如果要永久封禁，和第一个说明一样。
```

mode=Markdown　以MarkDown解析代码
mode=Markdownv2　以MarkDownV2解析代码
mode=html　以HTML解析浏览器页面代码
